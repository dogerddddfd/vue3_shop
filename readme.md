# 电商项目--豪华升级版 v3

* vue3重构

* 技术栈：vue3, element-plus, axios

* ./start一键运行

# 用法
### 依赖安装
* node,npm
* [docker](https://www.docker.com/)，[docker compose](https://docs.docker.com/compose/)
* node_modules安装
```sh
# vue_shop
   cd server
   npm i
   cd ../vue_shop_project
   npm i
```

### 启动
```sh
   ./start.sh
```

### 访问

* 127.0.0.1:5173
